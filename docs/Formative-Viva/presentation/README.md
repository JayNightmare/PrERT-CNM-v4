# Formative Viva Reveal.js Deck

This folder contains a standalone Reveal.js presentation for the formative viva.

## Files

- `index.html` - main slide deck
- `style.css` - custom presentation styling

## Run Locally

Option 1 (quick): open `index.html` directly in a browser.

Option 2 (recommended): serve the folder with a local HTTP server:

```bash
cd docs/Export/Formative-Viva/presentation
python3 -m http.server 8000
```

Then open: `http://localhost:8000`

## Presenter Tips

- Press `S` for speaker view.
- Use arrow keys to navigate slides.
- Use `Esc` for slide overview mode.
- Every slide now includes bullet-point speaker script notes in the Reveal speaker view.
